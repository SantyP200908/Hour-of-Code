Goals:

This project goal is to automate school daily activities for students and teachers.

Our project will be about a NFC system to take assistance of students, also it can be used like a key for student lockers, and it can be used for uploading report cards to the server of the school.

This can automatize a lot of the parts that student do on his daily lives on school like ones i mention before.

Auto taking list.

Private lockers with any tag.

Also you can have a special printer on the school where you can go and print the qualifications with your tag.

You can buy lunch on cafeteria with a previous charge in your tag.

And a lot of more thing that we can be improving if the project go like is planed are a lot of thing that we can do with this technology.
